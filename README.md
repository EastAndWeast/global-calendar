# 全球日历订阅器 (Global Calendar Hub)

一个聚合全球节假日、活动日历的订阅工具。

## 功能

- 🌍 支持按国家/地区订阅节假日日历
- 📱 一键生成 iCal / Google Calendar 订阅链接
- 🎯 分类浏览：节假日、金融市场、体育赛事等

## 使用方法

1. 打开 `index.html`
2. 选择你想订阅的日历
3. 点击订阅按钮，获取链接
4. 添加到你的日历应用中

## 日历来源

- 中国节假日
- 美国节假日
- 日本节假日
- 欧洲节假日
- 金融市场休市日历
- 重要体育赛事

## 技术栈

- 纯前端，无需后端
- 使用 Google Calendar API 和 iCal 格式
